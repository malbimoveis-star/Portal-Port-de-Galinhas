Politica de Privacidade

Esta Politica de Privacidade descreve como coletamos, usamos e protegemos os dados pessoais dos usuarios do Portal Porto de Galinhas.

1. Dados Coletados

Coletamos nome, e-mail e telefone de comerciantes cadastrados, alem de dados de navegacao anonimos para melhoria do servico.

2. Uso dos Dados

Os dados sao utilizados exclusivamente para exibicao dos anuncios, comunicacao com o comerciante e processamento de pagamentos.

3. Compartilhamento

Nao compartilhamos dados pessoais com terceiros, exceto quando necessario para processamento de pagamento (Mercado Pago).

4. Seguranca

Senhas sao armazenadas de forma criptografada (hash). Nunca solicitamos dados de cartao de credito diretamente.

5. Direitos do Usuario

O usuario pode solicitar a exclusao de seus dados a qualquer momento entrando em contato pelo canal de suporte.
